# Supplement Market Data Analysis

## Project Overview
This project analyses supplement-related product information to explore patterns in pricing, ingredients, product categories, and consumer-facing marketing claims. The goal is to demonstrate data cleaning, exploratory data analysis, visualisation, and evidence-based communication.

This project is for data analysis and science communication purposes only. It does not provide medical advice or make clinical recommendations.

## Motivation
I created this project because I run a health and supplement-related science communication channel. I wanted to use data analysis to better understand how supplement products are presented to consumers and how pricing, ingredients, and claims vary across products.

## Key Questions
- What are the common ingredients across different supplement products?
- How do prices vary across product categories?
- Are certain ingredients associated with higher prices?
- What types of marketing claims are commonly used?
- How can supplement-related information be communicated more clearly to consumers?
